# Gate.io MA10 趋势监控系统 v3.4

基于 **Gate.io U本位永续合约 API** 的 MA10（10周期移动平均线）趋势监控系统。支持 **Web 可视化面板** 和 **命令行脚本**，实时监控加密货币与美股代币的多周期趋势。

## 功能特点

### 核心监控
- **四周期趋势**：日K、4小时、60分钟、15分钟 MA10 趋势方向 + sparkline 走势小图
- **高流动性筛选**：自动过滤 24h 成交额 >= $5000万 的 USDT 合约，去噪低流动性标的；手动添加的标的不受影响
- **技术指标**：连续周期总涨跌幅（卡片展示）；MA10 + SAR（K线图叠加），趋势跟踪双重确认；SAR 方向基于算法内部状态判定（与币安等主流平台一致）
- **转折预警 v3.4**：SAR↔MA10 双向验证框架，两条触发路径 + MA10 三类型条件 + 中途提前确认机制（K线边界对齐调度 + MA10方向检测 + SAR同向验证 + 去重）+ 告警跨周期残留自动清理；前端周期/信号分类筛选 + 成交额/涨跌幅展示 + 点击卡片添加持仓/查看K线
- **K线图表**：点击卡片弹出交互式 K线图（TradingView lightweight-charts），支持周期切换 + MA10/SAR 指标叠加

### 预警与通知
- **转折预警**：SAR↔MA10 双向验证，四周期独立检测。类型1(MA10拐头2周期+开盘价确认)、类型2(MA10刚拐头+量能爆发)、类型3(MA10刚拐头+量递增)。点击周期格子弹出菜单订阅企业微信推送
- **中途确认**：类型1 在下一K线半程点调度单标数据获取（对齐K线边界，非简单延时），验证类型1条件 + SAR同向确认，写入 tp_state 去重
- **极偏信号 v4.1**：MA10 连续≥5 + SAR 连续≥5 同向时，四条件检测极端偏离。分周期阈值（日K K=2.5/4h K=3.0/1h K=4.0）。**仅在每周期收盘前最后一次刷新时推送**（精简汇总格式：标的名 + 周期 + 极多/极空），去重绑定 candle 起始时间
- **价格提醒**：每标的可设置突破/跌破价格阈值，触发后高亮脉冲闪烁

### 交互与体验
- **SSE 实时推送**：后端数据刷新后主动推送到前端，60s 轮询降级备用
- **亮色/暗色主题**：一键切换，localStorage 持久化
- **骨架屏加载**：刷新时 shimmer 动画占位卡片
- **键盘快捷键**：`/` 搜索 `R` 刷新 `Esc` 关闭
- **持仓管理**：多/空/无三态标记，服务器端持久化，持仓标的最大置顶
- **标的管理**：输入简称（如 `btc`）自动推导完整合约名，实时验证 Gate.io 合约是否存在，无需修改 JSON 文件
- **搜索 + 排序**：实时搜索过滤，按持仓/成交额/涨跌幅多维度排序
- **企业微信订阅推送**：点击周期格子 → 弹出菜单 → 选择「企业微信预警」，触发转折信号时一次性推送，推送后自动取消订阅。支持一键重置全部订阅
- **移动端适配**：底部导航栏，响应式网格布局，手机/电脑均可使用

### 性能与架构
- **模块化架构**：app.py / monitor.py / alerts.py / state.py / templates/index.html 五模块独立
- **SQLite K线缓存**：历史 K线本地持久化，API 调用量减半（500 → ~200 次/刷新），WAL 模式提升并发
- **Gzip 压缩**：JSON 响应自动压缩，/api/data ~150KB → ~25KB，节省 85% 带宽
- **SOCKS5 代理**：仅 Gate.io API 走代理通道，不影响系统网络
- **连接池复用**：线程级 requests.Session 连接池，避免重复 TCP 握手
- **原子写入**：状态文件 `tempfile + os.replace` 防止崩溃损坏
- **API 限流保护**：全局信号量（Semaphore 5）+ 请求延迟，稳定 ~10 req/s；类型检查 + 3 次重试退避

## 快速开始

### 安装依赖



```bash
python gateio_futures_monitor.py
```

### 转折预警 (v3.0+)

SAR↔MA10 双向验证框架，两条触发路径互为验证：

| 路径 | 触发顺序 | 验证方式 |
|------|----------|----------|
| 路径 A | SAR 先翻转 | 验证 MA10 是否满足三类型条件之一 |
| 路径 B | MA10 先满足 | 验证 SAR 是否已同向翻转 |

MA10 转折三类型（多头为例，空头对称相反）：

| 类型 | 条件 | 含义 |
|------|------|------|
| 类型 1 | 连续转折周期 = 2 + 当前价 > MA10 + 开盘价 > 上周期开盘价 | MA10 明确拐头，最稳 |
| 类型 2 | 连续转折周期 = 1 + 当前量 > 10周期均量 + 当前价 > MA10 | MA10 刚拐头 + 量能爆发 |
| 类型 3 | 连续转折周期 = 1 + 当前量 > 上周期量 > 上上周期量 + 当前价 > MA10 | MA10 刚拐头 + 量递增 |

**类型 1 中途确认 (v3.2 修复)**：连续=1 且不满足类型 2/3 时，在下个K线半程点调度单标获取（调度时间对齐K线边界，方向以MA10趋势为准，执行时验证SAR同向 + tp_state去重）。

## 注意事项

1. **Gate.io 合约价格 ≠ 真实股价**：美股代币合约存在溢价，但涨跌趋势与真实美股高度一致
2. **新合约数据不足**：部分新上线合约日K历史数据较少，需等待积累
3. **局域网访问**：默认绑定 `0.0.0.0`，同 WiFi 可访问；跨网络需云服务器部署

## SOCKS5 代理配置

如需通过代理访问 Gate.io API（不影响系统网络），编辑 `monitor.py`：

```python
# 设置代理地址（V2Ray/Xray/Clash 等）
PROXY_URL = "socks5://127.0.0.1:10808"

# 不走代理则设为 None
PROXY_URL = None
```

代理仅对 Gate.io API 生效，系统其他网络请求不受影响。需安装 PySocks：`pip install PySocks`
